# إسلام ريلز 🌙

صانع ريلز قرآنية مع تصدير MP4

## الرفع على Railway (مجاناً)

### الخطوات:

1. **إنشاء حساب على GitHub**
   - روح على [github.com](https://github.com) وسجل
   - ابعت New Repository → اسمه `islam-reels`
   - ارفع كل الملفات دي فيه

2. **ربط Railway بـ GitHub**
   - روح على [railway.app](https://railway.app)
   - سجل بحساب GitHub
   - اضغط **New Project** → **Deploy from GitHub repo**
   - اختار الـ repo بتاعك `islam-reels`
   - Railway هيشتغل تلقائياً ويديك رابط

3. **الرابط**
   - بعد الـ deploy هتلاقي رابط زي:
   `https://islam-reels-production.up.railway.app`
   - شاركه مع أي حد!
